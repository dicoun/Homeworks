Тема итогового проекта: 'Онлайн тест для школьников разных классов по курсу математики'
Цель: Определение уровня владения предметом.
Планируемый функционал: 

1) В приложении предусмотрена авторизация для администратора;
2) При входе в приложение и нажатии кн."Пройти тест" пользователю будет предложено заполнить свои реквизиты (ФИО и номер класса обязательно);
3) Далее, в зависимости от выбранного класса, по каждой из тем программы пользователю будут предложены задачи в виде теста;
4) Для каждой задачи на странице будет предусмотрен таймер;
5) По итогу теста пользователю будет выставлена оценка;
6) При авторизации администратор сможет просмотреть журнал сдачи теста, в котором будут отражены реквизиты пользователей, прошедших тестирование, дата и их балл;
7) Также, по каждому прошедшему тест пользователю будет предусмотрена возможность расшифровки результатов (просмотр задач, попавших в тест, ответов
пользователей и правильных ответов);
8) Будет предусмотрена возможность просмотра общей статистики сдачи по классам, по темам, по времени (графики);
